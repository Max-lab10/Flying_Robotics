# Flying_Robotics
Командная задача НТО по профилю "Летающая робототехника" команды "Полетели"

Клонируйте репозиторий в домашнюю директорию:
```bash
cd ~
git clone https://github.com/Max-lab10/Flying_Robotics.git
```

Перейдите в папку проекта и запустите скрипт инициализации:
```bash
bash start.sh
```
Создайте текстовый документ, вставьте туда код сайта, сохраните, поменяйте формат txt на html.
